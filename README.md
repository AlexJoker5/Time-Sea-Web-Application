# Time Sea Web Application

*This web application is developed by using Laravel Framework and Mysql DB server.*

**Me and My friend, Daniel developed this application developed this application for the school project.**


In this web application, we focus on cart feature. User can add watches that they like into carts and can also remove them. The total price after reducing tax will be calculated automatically.
